- [Mobile-first CSS rule](mobile-first.md) — All CSS base styles target mobile; scale UP with min-width queries. Never max-width overrides. User confirmed this is required.
- [Anti-slop token decisions](anti-slop-tokens.md) — Instrument Serif banned; --font-display = Outfit. bg = #F5F5F4. No uppercase+letterSpacing on any public page label. No borderRadius:9999 — badges use 4px, chips use 6px. No violet/purple glow box-shadow anywhere. Primary CTAs and toggles use #111 dark, not brand accent.
- [Primary CTA color rule](cta-color.md) — All primary CTA buttons on public pages use #111 dark background, not var(--color-accent). Brand accent #6C47FF reserved for logo, text highlights, small badges only. Pricing toggle indicator also #111.
- [Landing page architecture](landing-arch.md) — Split hero (copy left, doc mockup right), dark stats strip, pipeline strip, numbered feature table, competitor grid, dark CTA. Each section must have a distinct visual identity.
- [Auth page layout](auth-layout.md) — Split panel: dark brand panel left (pipeline viz, metrics strip, testimonial, compliance badges), light form panel right. Auth CSS in shell.css. Breakpoint: min-width 900px shows brand panel.
- [Shell CSS duplicate sidebar](shell-duplicate.md) — shell.css has two .sidebar definitions (~line 29 dark hardcoded, ~line 1332 CSS-vars). Studio sidebar renders correctly; do not attempt to merge — risk of breakage outweighs benefit.
- [Public pages architecture](public-pages-arch.md) — All marketing pages use shared PublicHeader + PublicFooter from @/components/layout/. Pages in main/src/pages/public/. Router has real routes for /pricing /templates /about /support plus proper 404 (no more * → Navigate to /).
- [Currency rule](currency-rule.md) — ALL pricing is ₦ Naira only. No $ anywhere. Nigeria-only product. Primary display is ₦ amount, no "≈ ₦" approximation lines.
- [Templates fill engine](templates-fill-engine.md) — PublicTemplatesPage has a full modal (TemplateModal) with 2-panel layout: left=form fields, right=LiveDocPreview updating live. Clicking "Use Template" shows inline auth prompt (sign up / log in). Each template has typed formFields + previewFn. All 16 templates have real fields.
- [Width shaking fix](width-shaking.md) — Fixed by adding overflow-x:hidden + scrollbar-gutter:stable to html in globals.css, plus overflow-x:hidden on body. Prevents scrollbar appearing/disappearing from shifting content width.
- [Mobile input zoom fix](mobile-input-zoom.md) — ALL inputs globally have font-size:16px in globals.css. iOS zooms on focus when font-size < 16px. This is a global rule — never override inputs to font-size < 16px in public pages.
