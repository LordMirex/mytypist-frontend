## Strategic Aim
The goal is to successfully extract and merge the best features, components, and logic from both `v1` and `v2` to create a complete and optimized final frontend.

## Roadmap
1. **Extraction**: Identify core assets and logic from `v1` and `v2`.
2. **Merging**: Integrate extracted pieces into this `main` directory.
3. **Finalization**: Once the unified version is stable and complete, it will be moved to the root folder.
4. **Cleanup**: Legacy directories `v1` and `v2` will be deleted.
