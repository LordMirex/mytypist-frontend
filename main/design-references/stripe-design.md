# Stripe Design System Reference

Source: https://github.com/VoltAgent/awesome-design-md (Stripe DESIGN.md)

## Colors

- **Primary:** `#533afd` (indigo CTA)
- **Primary Deep:** `#4434d4`
- **Primary Press:** `#2e2b8c`
- **Primary Soft:** `#665efd`
- **Ink:** `#0d253d` (deep navy body text)
- **Ink Secondary:** `#273951`
- **Ink Mute:** `#64748d`
- **Canvas:** `#ffffff`
- **Canvas Soft:** `#f6f9fc`
- **Canvas Cream:** `#f5e9d4`
- **Hairline:** `#e3e8ee`
- **Hairline Input:** `#a8c3de`
- **Ruby:** `#ea2261`
- **Magenta:** `#f96bee`
- **Brand Dark 900:** `#1c1e54` (dashboard dark shell)

## Typography

- **Font:** Sohne (proprietary) → fallback to Inter at weight 300
- **Weight:** 300 (thin) across all display tiers
- **Negative tracking:** -1.4px at 56px down to -0.2px at 20px
- **Feature:** `ss01` globally, `tnum` on money cells
- **Display XXL:** 56px / 300 / 1.03 / -1.4px
- **Display XL:** 48px / 300 / 1.15 / -0.96px
- **Display LG:** 32px / 300 / 1.1 / -0.64px
- **Body MD:** 15px / 300 / 1.4 / 0

## Components

- **Buttons:** Pill shape (9999px), padding 8px 16px, weight 400
- **Cards:** `rounded.lg` (12px), padding 32px, hairline border
- **Inputs:** `rounded.sm` (6px), padding 8px 12px
- **Nav:** Transparent over gradient, white on scroll

## Key Patterns

- Gradient mesh backdrop on every marketing hero
- Single indigo CTA per band
- Dashboard dark shell (Brand Dark 900)
- Tabular figures for money/numerics
- Composited product UI mockups instead of photography
