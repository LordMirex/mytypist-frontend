# User Journey Maps: The Document Pipeline

This document visualizes the spatial journey of a user through the MyTypist **Document Operating System**.

## Journey 1: Guest Conversion Flow
*Objective: Demonstrate immediate value (Formatting Fidelity) before requiring account creation.*

1.  **Editorial Gateway (Home)**: User enters a high-density, professional home environment.
2.  **Live Selector**: User selects a template from a typography-focused grid (e.g., "Standard Service Agreement").
3.  **The Studio (Pre-Auth)**: 
    - User enters the high-precision studio.
    - Fills in 2-3 key fields (e.g., Client Name, Date).
    - **Live Preview** instantly shows the document with perfect formatting.
4.  **Fidelity Unlock (Finalize)**: 
    - User clicks "Finalize & Download".
    - System triggers a "Fidelity Preservation Account" modal.
    - User signs up to save the document and unlock the PDF export.

## Journey 2: Operational Pro Flow (The Main Workspace)
*Objective: Facilitate high-speed document operations for power users.*

1.  **The Command Dashboard**: 
    - User lands in a persistent, spatial workspace.
    - **Pipeline Visualization** shows 3 documents pending approval.
2.  **Rapid Drafting**:
    - User opens "Service Proposal" from the dashboard.
    - Enters the Studio using **Keyboard-First** shortcuts to tab through fields.
    - Uses **Inline Styling** to bold a specific clause in the live preview.
3.  **Operational Handoff**:
    - User changes status to "Awaiting Approval."
    - **Optimistic Update** instantly moves the document to the next stage in the pipeline.
    - Spatial transition back to the dashboard to pick up the next task.

## Journey 3: Infrastructure Audit (Admin)
*Objective: Ensure platform-wide document craftsmanship and system health.*

1.  **Monitor Gate**: Admin checks the **Fidelity Accuracy Score** gauge.
2.  **Template Governance**:
    - Admin reviews a new community-uploaded template.
    - Uses the **Structure Inspector** to confirm 100% Visual Parity.
    - Audits placeholder tags for naming consistency.
3.  **Approval**: Admin approves template; it instantly appears in the spatial Gallery for all users.
